# Quick Setup Guide

## 1. Clone the Repository
```bash
git clone <your-repo-url>
cd mern
```

## 2. Install Dependencies
```bash
npm install
```

## 3. Start the App
```bash
npm run dev
```

Open the Vite URL, usually http://localhost:5173.

## 4. Try the Main Flows
- Sign up as a user
- Create a donation
- Sign in as an NGO user and claim it
- Sign in as a delivery partner and complete the order

## Notes
- No backend server is required.
- No MongoDB setup is required.
- The browser store seeds demo data if localStorage is empty.
